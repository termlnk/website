# Keybindings

Termlnk comes with a full set of default keyboard shortcuts and lets you customize every binding.

## Default Keybindings

> On macOS, `Ctrl` is replaced by `Cmd` unless noted otherwise.

### General

| Action | macOS | Windows / Linux |
| :--- | :--- | :--- |
| Command palette | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Settings | `Cmd+,` | `Ctrl+,` |
| Toggle sidebar | `Cmd+B` | `Ctrl+B` |
| Toggle full screen | `Ctrl+Cmd+F` | `F11` |

### Terminal

| Action | macOS | Windows / Linux |
| :--- | :--- | :--- |
| New terminal | `Cmd+T` | `Ctrl+Shift+T` |
| Close terminal | `Cmd+W` | `Ctrl+Shift+W` |
| Split horizontal | `Cmd+D` | `Ctrl+Shift+D` |
| Split vertical | `Cmd+Shift+D` | `Ctrl+Shift+E` |
| Focus next pane | `Cmd+]` | `Ctrl+]` |
| Focus previous pane | `Cmd+[` | `Ctrl+[` |
| Magnify pane | `Cmd+Shift+Enter` | `Ctrl+Shift+Enter` |
| Buffer search | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| Clear buffer | `Cmd+K` | `Ctrl+Shift+K` |

### SSH & SFTP

| Action | macOS | Windows / Linux |
| :--- | :--- | :--- |
| Open host manager | `Cmd+Shift+H` | `Ctrl+Shift+H` |
| Quick connect | `Cmd+Shift+C` | `Ctrl+Shift+C` |

### AI Agent

## Customization

Override any keybinding in **Settings > Keybindings**. The editor shows all registered commands with their current bindings.

To customize:

1. Open **Settings > Keybindings**.
2. Search for the command you want to rebind.
3. Click the current binding and press your desired key combination.
4. The new binding saves automatically.

Custom bindings take priority over defaults. To reset a binding, click the reset icon next to it.

> **Tip** — you can assign multiple key combinations to the same command, or leave a command unbound by removing its binding.
