# 主题与外观

Termlnk 提供丰富的主题定制能力，覆盖终端和应用界面的各个方面。

## 内置主题

预装 **71 套主题**：

- 56 套深色主题
- 15 套浅色主题

在设置中一键切换，即时生效。

## 实时主题编辑器

使用内置的实时主题编辑器，可视化调整颜色并实时预览效果，无需手动编辑配置文件。

## Base46 颜色体系

Termlnk 使用 Base46 颜色系统。每个主题由一组基础颜色变量定义，覆盖：

- 应用界面颜色（背景、前景、边框、高亮等）
- 终端 ANSI 16 色
- 终端扩展色

## 终端配色定制

独立于应用主题，可单独调整终端的：

- 前景色 / 背景色
- ANSI 16 色映射
- 光标颜色
- 选区颜色

## 应用界面定制

可定制应用外壳（Chrome）的各类颜色，包括：

- 标题栏
- 侧边栏
- 面板背景
- 边框和分隔线
- 按钮和控件

## 透明与模糊

| 设置 | 说明 |
| --- | --- |
| 窗口透明度 | 调整整体窗口的不透明度 |
| 背景模糊 | 启用毛玻璃效果（需系统支持） |

## 字体设置

- 终端字体族
- 字体大小
- 行高
- 字重
- 连字开关

## 自定义 CSS

支持注入自定义 CSS，对应用界面进行深度定制。
