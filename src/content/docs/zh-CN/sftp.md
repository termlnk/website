# SFTP 文件管理

Termlnk 内置双栏 SFTP 文件浏览器，与 SSH 主机配置共享连接。

## 双栏浏览器

左栏为本地文件系统，右栏为远程文件系统。两侧均支持目录浏览、文件预览和常规文件操作。

## 拖拽传输

在本地和远程栏之间直接拖拽文件或文件夹即可发起传输。

## 传输队列与历史

- **传输队列** — 查看正在进行和排队中的传输任务
- **传输历史** — 查看已完成的传输记录

## 权限编辑器

选中远程文件后可直接编辑文件权限（读 / 写 / 执行），无需手动执行 `chmod`。

## 批量操作

支持多选文件进行批量操作：

- 批量下载 / 上传
- 批量删除
- 批量移动

## 文件夹与重命名

- 在本地或远程创建新文件夹
- 右键重命名文件或文件夹

## 右键菜单

文件右键菜单提供常用操作：下载、上传、删除、重命名、复制路径、编辑权限等。

## 连接到 SSH 主机

SFTP 浏览器复用 SSH 主机配置中的连接信息。在主机树中选择已配置的主机，即可直接打开 SFTP 会话，无需重复配置。
